Atavistic Sanctum

Gamemaker Studio 2

Github https://github.com/BladedTaco
itch.io https://bladedtaco.itch.io/

Year 12 Software Development folio project made in 2018, with an attempted revival in 2019 through a hitbox creator tool that addressed the problem of maintaining over 1000 hitboxes as raw numbers.

The project is an in-depth platform fighter including:
  • 4 playable characters
  • up to 8 concurrent players
  • 3 stages
  • Interactive Replays (take control at any time)
  • Online Play
  • Training Mode
  • Directional Grabs and Throws
  • Stamina-based Shielding and Rolls
  • tilt, smash, aerial, special, and ledge attacks
  • totalling in over 200 actions!


    • The hitbox creator makes hitbox/hurtbox creation and maintenance much easier thanks to the organization, visualization, and rapid prototyping it provides. I also included parsing and saving of hitboxes to directly change the Atavistic Sanctum game files to save any copy-pasting hitboxes between the tool and the game and prevent any errors that could arise from human error.
    Screentone Dot Shader
    (GML) Exceptionally Clean Code

    Screentone Dot Shader

    Gamemaker Studio 2.3 Beta

    Github https://github.com/BladedTaco
    itch.io https://bladedtaco.itch.io/

    A simple project from 2020 made alongside the release of Gamemaker Studio 2.3 beta which introduced lightweight classes.

    The project displays a simple screentone dot shader which is controlled by the on-screen tokens. The project revolves around communication between the CPU-based GML code and the GPU-based GLSL shader, which is made highly readable and simple by the introduction of the simple classes.

    While not an impressive project, it showcases the ability of the Object-Oriented Programming paradigm to excel when working to its strengths; a programming paradigm that many like to eschew and make mockery of with purposely ill-fitting examples.
    Spell Programmer
    (GML) Visually Interesting Visual Programming Language

    Spell Programmer

    Gamemaker Studio 2

    Github https://github.com/BladedTaco
    itch.io https://bladedtaco.itch.io/

    My first proof-of-concept of my dream project made in 2020: A dueling game where spells are fully programmable from scratch, render into human-readable spell-circles, and anyone can learn how to program spells.

    The proof-of-concept focusses on the spell programming side, where a stacked-hex-grid-graph is used to hold spell tiles which can be piped together into a central trick tile that affects the world. I achieved what I wanted with this prototype, in that I found out what I needed to learn to make my dream game a reality.

    This project led me into the fields of graphics programming, functional programming, visual language design, human computer interaction, and computing education. All of which I continue to learn more about in order to one day return to this game. Most notably, this project was the impetus for my Honours Thesis Project.
    Honours Project
    (UE4) Runtime Haskell as C++ Types

    Honours Thesis Project

    Unreal Engine 4.27 (Custom)

    Github https://github.com/BladedTaco
    itch.io https://bladedtaco.itch.io/

    My 2023 Honours Thesis Project made in a custom version of Unreal Engine 4.27 which brings back the HTML export option. Currently targeting official publication at a conference.

    My Thesis: "Leveraging Familiar Game Concepts to Create a Game for Teaching Functional Programming" is aimed at addressing the hardest part of turning my dream game into fruition, teaching people how to program. The hypothesis, is that many people already know programming-like systems from learning game strategies and that all that is needed is to translate this into concrete concepts and build confidence.

    The game itself required a highly complex haskell-like type system coded entirely in runtime C++. Careful design tricks and a custom pre-processing pipeline was used to ensure this type system was type safe, while also being type erased. Don't know what that means? contact me! I love talking about this project!
    Bounce Bounce Jonathan
    (UE5) Slapstick Physics-Based Obstacle Course Racer

    Bounce Bounce Jonathan

    Unreal Engine 5.3

    itch.io https://bladedtaco.itch.io/

    A collaborative university project developed over 6 weeks in 2023. Due to my university's plagiarism guidelines source code is not available.

    On this project, I primarily focussed on the camera/physics interactions, realtime world-space 3D textures and materials, and screen-space effects. I am also the voice of the entirely acapella sound design, which has the strange effect of draining sanity through humour.

    The main challenges I solved revolve around having a physics-based movement system work alongside the state-based character movement component that UE5 requires characters to use. This was solved mostly by overriding methods into no-op functions, hooking into available event delegates, and careful tweaks to real-world physics formulas. I recommend others to do this too if they wish to learn how the inbuilt movement system works under-the-hood.
